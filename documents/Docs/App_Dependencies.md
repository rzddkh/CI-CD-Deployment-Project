# Backend dependencies

- Node.js
- Express.js
- Postgres database
- dotenv
- sequelize
- bcrypt
- You need an env file to store enviroment variables

## dotenv Reference

- POSTGRES_USERNAME=`value`
- POSTGRES_PASSWORD=`value`
- POSTGRES_HOST=`value`
- POSTGRES_DB=`value`
- AWS_BUCKET=`value`
- AWS_REGION=`value`
- AWS_PROFILE=`value`
- JWT_SECRET=`value`
- URL=`value`
- DB_PORT=5432
